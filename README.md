This repository contains an **Exploratory Data Analysis (EDA)** project performed on the famous **Titanic dataset**.

## Objectives
- Perform EDA on a real-world public dataset
- Handle missing and inconsistent data
- Visualize important relationships using graphs
- Identify factors affecting passenger survival
